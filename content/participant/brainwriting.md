# Brain Writing

Instead of calling ideas out, you build quietly on prompts — each new screen
hands you someone else's idea as a springboard.

## How to take part

- Read the provocation you're given and add an idea that builds on it.
- You'll be handed different starting points as you go, so everyone's thinking
  cross-pollinates.

## Tips

- "Yes, and…" — extend the prompt rather than starting from scratch.
- Quiet and steady beats loud and fast here; it's designed so everyone
  contributes equally.
