# Value / Doability

Score each idea on two questions: how much value would it create, and how doable
is it? The scores plot each idea on a grid.

## How to take part

- For each item, set its **value** and its **doability**.
- The scatter plot shows where every idea lands, coloured by how much the group
  agrees.

## Reading the grid

- **High value, high doability** (top-right) — quick wins worth doing now.
- **High value, low doability** — big bets, worth it if you can tackle the
  obstacles.
- **Low value**, wherever it sits, is usually easy to set aside.
