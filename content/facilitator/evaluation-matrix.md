# Evaluation Matrix — for facilitators

The Evaluation Matrix compares a set of options against several **criteria** at
once. Each option gets a score on every criterion; the tool averages the group's
scores and shows a matrix with a total per option, so you can see not just which
option is strongest overall but *why* — where each one is strong or weak.

Its purpose isn't to crown a single winner. Use it to understand the field, then
combine, modify, or sequence the strongest options.

## Set it up

- Pipe in the options to evaluate (for example, the shortlist from a vote).
- In the tool's settings, list your **criteria** (comma- or line-separated), e.g.
  "Impact, Effort, Fit". Set the **rating scale** maximum if 1–5 isn't right.
- Set the criteria *before* scoring starts — changing them mid-way detaches
  earlier scores.

## How participants score

Participants rate **one criterion at a time**: all options on the first
criterion, then "Next →" to the second, and so on. Short screens, and everyone
uses the same yardstick for each criterion.

## Read the results

The matrix shows each option's average on every criterion, a **Total** (the mean
of its criterion averages), and how many people scored it. A small "±" is the
spread — a large spread means the group disagreed on that cell, which is often
the most interesting thing to discuss.

## Tips

- Best with a handful to a dozen options and a few clear criteria.
- Phrase criteria so higher always means better (e.g. "Ease" not "Difficulty").
- Follow the matrix with a discussion of the high-spread cells.
