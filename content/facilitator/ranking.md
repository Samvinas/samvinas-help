# Ranking — for facilitators

Ranking orders items by preference. Unlike voting, everyone ranks every item, so
it shows both the group's order **and** where opinion is split.

## How to run it

- Pipe in a short list — ranking works best with roughly 5–10 items; long lists
  are tiring.
- Set the criterion in the **Prompt**.
- Watch the average ranks and the disagreement colouring.

## Facilitation tips

- High disagreement on an item is a conversation worth having — surface it.
- Keep the list short. If it's long, **Vote** first to cut it down, then rank the
  finalists.
