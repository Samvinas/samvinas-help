# Instructions — for facilitators

The Instructions tool shows participants a page of your own Markdown — a brief, a
task, or a set of ground rules. It takes no input; it's purely for telling people
what to do.

## How to run it

- Add it to a flow, open its ⚙ **Settings**, and write your Markdown in the
  editor (with live preview), then **Focus** it.
- It's especially useful inside **Breakout Groups** and **Personal Workspaces**,
  where each group needs its own standing instructions while they work.

## Facilitation tips

- Use Instructions for *this session's* task; use the tool **?** links (these
  help pages) for standing guidance that doesn't change.
- You can edit the text mid-session — participants see the update live.
