# Word Cloud — for facilitators

Word Cloud builds a live concordance from the text of piped items, sizing words
by frequency. A fast, visual way to surface themes and shared language.

## How to run it

- Pipe in a text-rich tool (Brainstorm, Rich Brainstorm).
- Project it on the **Display** for the room.

## Facilitation tips

- Use it as a talking point: "these words keep coming up — what does that tell
  us?"
- It reflects language, not judgement; pair it with a scoring tool when you need
  to prioritise.
