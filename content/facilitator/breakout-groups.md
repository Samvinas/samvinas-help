# Breakout Groups — for facilitators

Breakout Groups splits participants into small teams, each with its own shared
workspace built from a template. You can hand each group a slice of items or the
whole set, and gather results back to the parent event.

## How to run it

- Set the number of groups and build the group template.
- Choose how items are divided (a slice each, or the full set to every group).
- While groups work, add an **Instructions** tool to give them standing guidance;
  use a group's **Edit** button if one needs a different flow.
- Use **Gather Results** to pull a chosen tool's output from each group back into
  the main flow.

## Facilitation tips

- Small groups dramatically raise participation — ideal for large sessions.
- Give a clear task and a time box; the parent **Timer** stays visible inside
  groups.
