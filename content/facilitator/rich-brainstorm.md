# Rich Brainstorm — for facilitators

A richer Brainstorm: multiline Markdown plus a freehand `/sketch` canvas. Use it
when ideas need more than a single line — concepts, sketches, structured notes.

## How to run it

- Set an open **Prompt**, as with Brainstorm.
- Let people know they can sketch with `/sketch`.

## Facilitation tips

- Ideal for design, systems, or otherwise visual problems.
- Consuming tools (Voting, Ranking) show a "more" button, so rich ideas stay
  readable when you converge.
