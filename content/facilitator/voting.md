# Voting — for facilitators

Voting is a fast convergence tool: a star toggle that surfaces the group's
favourites in real time.

## How to run it

- Pipe in the items to vote on (usually from Brainstorm or Clustering).
- Set the **max votes per person** in settings — a good rule of thumb is about a
  third of the number of items.
- Put a clear criterion in the **Prompt** ("Which three would have the biggest
  impact?").
- Watch the live ranking; **Lock** when voting's done.

## Facilitation tips

- Say the criterion out loud — people vote very differently for "most exciting"
  vs "most feasible."
- Voting is a temperature check, not a verdict. Use it to narrow the field, then
  discuss the top items.
- For a criteria-rich decision, follow voting with Value / Doability or Ranking.

## Piping onward

Voting adds vote counts to items, so a downstream tool can filter by a minimum
number of votes (e.g. only items with two or more votes flow into Value /
Doability).
