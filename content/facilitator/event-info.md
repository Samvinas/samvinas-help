# Event Info — for facilitators

Event Info is a standing page inside your event for the practical details
people keep asking about: WiFi credentials, the day's schedule, room
locations, lunch arrangements. Write it once; participants find it in their
menu whenever they need it, without interrupting the current activity.

## How to run it

- Enable Event Info in the **Utilities** section of your session view.
- Write the content in Markdown — headings, lists and links all work.
- Rename the page's label if you like, and add more than one instance when a
  single page would get long (say, *Schedule* and *Venue*).
- Participants open it from their menu at any time; it doesn't take part in
  the flow or need Focus.

## What belongs here

WiFi and passwords · schedule and timings · rooms and maps · meals and
dietary notes · emergency contacts · anything you'd otherwise put on a
flip-chart by the door. For **task** instructions tied to a step in the work,
use the [Instructions](/facilitator/instructions.html) flow tool instead —
Event Info is for the standing facts of the event.
